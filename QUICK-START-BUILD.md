# Quick Start: Build Website Now

> *Start building in 5 minutes. No philosophy, just code.*

---

## Setup (5 minutes)

```bash
# 1. Install dependencies (if not already)
npm install

# 2. Verify environment
cp .env.example .env.local
# Add your API keys (Supabase, SendGrid, etc.)

# 3. Start development server
npm run dev
```

**Open:** http://localhost:3000
