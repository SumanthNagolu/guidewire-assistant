## Responsibilities

1. **Design User Experience**
   - Map user journeys
   - Design information architecture
   - Create wireframes
   - Design interaction patterns
   - Ensure accessibility

2. **Design User Interface**
   - Create design system
   - Design all screens
   - Specify colors, typography, spacing
   - Create high-fidelity mockups
   - Maintain design consistency

3. **User Research**
   - Understand user needs
   - Conduct usability testing
   - Gather feedback
   - Iterate on designs

4. **Collaboration**
   - Work with BA on requirements
   - Guide developers on implementation
   - Review implemented UI
   - Ensure design fidelity
