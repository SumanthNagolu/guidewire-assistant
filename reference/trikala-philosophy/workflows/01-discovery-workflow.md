# Discovery Phase Workflow

## Objective
Extract complete, unambiguous vision from CEO through exhaustive questioning.

## Active Personas
- **Saraswati** (Vision Extractor) - PRIMARY
- **CEO Sumanth** (Visionary) - COLLABORATOR
- **Sadhguru** (Gatekeeper) - REVIEWER

## Duration
Until vision is complete (typically 2-4 weeks for 2000 questions)

## Workflow Steps

### Step 1: Initiation
**Persona:** Sadhguru  
**Action:** Bless the discovery process, set quality standards

### Step 2: Question Preparation
**Persona:** Saraswati  
**Action:** Prepare 2000-question framework across 10 categories

### Step 3: Vision Extraction Sessions
**Personas:** Saraswati + CEO  
**Process:**
1. Saraswati asks questions (Socratic method)
2. CEO provides answers
3. Saraswati digs deeper (5 Whys)
4. Document every insight
5. Surface all assumptions
6. Repeat until category complete

**10 Categories:**
1. Core Value Proposition (200 questions)
2. User Personas (200 questions)
3. User Journeys (200 questions)
4. Business Model (300 questions)
5. Technical Constraints (200 questions)
6. Success Metrics (100 questions)
7. Risk Scenarios (200 questions)
8. Organization & Process (150 questions)
9. 5-Year Vision (150 questions)
10. Edge Cases & Details (300 questions)

### Step 4: Documentation
**Persona:** Saraswati  
**Deliverables:**
- Complete Vision Document
- Stakeholder Analysis
- Requirements Catalog
- Risk Register

### Step 5: Review & Approval
**Persona:** Sadhguru + CEO  
**Check:**
- Can write 50-page spec with zero assumptions? ✅
- Developer can build with zero questions? ✅
- All stakeholders identified? ✅
- All use cases documented? ✅
- CEO confirms understanding? ✅

### Step 6: Handoff
**From:** Saraswati  
**To:** Krishna (Architect) + Product Owner  
**Context:** Complete vision document, all requirements, all constraints

## Success Criteria
✅ 2000+ questions asked and answered  
✅ Zero ambiguities remain  
✅ CEO says "You understand my vision completely"  
✅ Sadhguru approves for clarity and alignment  
✅ Next personas can proceed without questions
